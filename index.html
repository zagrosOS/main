What is Zagros OS?

Zagros OS is a revolutionary operating system built from scratch in Assembly. It aims to be fast, lightweight, and optimized for high performance.

Key Features

  • 🔹 Ultra-fast performance with low resource usage
  • 🔹 Built entirely in Assembly
  • 🔹 Custom kernel for optimized hardware interaction
  • 🔹 Support for multiple file systems
  • 🔹 Security-focused design
